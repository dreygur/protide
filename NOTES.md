## UI

1. The environment section is a bit confusing. Right now, clicking on an environment folder selects it instead of opening it. There should be a separate expand/open icon beside the folder name for opening folders.

2. Small icons across the UI should have proper hover effects so users can clearly understand which element they are pointing at.

3. Icons should also have tooltips/toolbar text to explain their purpose.

4. The grey text is difficult to read in some places. Also, the half-circle icon is a bit confusing.

## Request Body Tab

1. When switching to the Binary tab, the file browse option appears correctly, but the other body tabs disappear. There should still be a way to switch back to the other tabs.

## Response

1. Response tabs currently do not have a scrollbar. A scrollbar is needed there.
